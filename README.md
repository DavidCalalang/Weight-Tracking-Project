# Weight-Tracking-Project
I've been keeping track of my weight for fitness purposes for the past few years. I used to do so through paper notes, but since I've started college I used Google Keeps. This project will utilize the data kept from Google Keeps to perform exploratory analysis.

THe main tools used within this project are the libraries BeautifulSoup (for html file manipulation), pandas (for data cleaning and prep) and matplotlib (for data visualization).

Some future additions that could be made to this project are...
- Automating the collection of data (possibly using the Google Keeps API to run periodic jobs in the form of a data pipeline)
- Applying a neural network to make predictions of weight fluctuations in the future, however, this would require much more data to allow for the proper training and validation of the machine learning model.
